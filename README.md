# homebrew-espada

Homebrew tap for [Espada](https://github.com/saifaldin14/Espada-Auto) — a
self-hosted infrastructure control plane with a conversational AI surface.

## Install

```bash
brew tap saifaldin14/espada
brew install espada
```

## Update

```bash
brew update
brew upgrade espada
```

## Status

This tap is the supported install path until Espada lands in `homebrew-core`.
The formula installs a pre-built per-platform tarball — no network access
is required during `brew install`.
